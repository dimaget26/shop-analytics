# Анализ продаж интернет-магазина

## Описание проекта
Анализ 9 000+ заказов интернет-магазина за 2025 год + динамика за январь 2026.  
Рассчитаны ключевые метрики, выявлены проблемы с выручкой, даны рекомендации бизнесу.

## Ключевые выводы
- 📉 **Проблема:** Падение выручки в январе на ~100 млн руб. по сравнению со среднемесячной
- 🛒 **Причина:** Снижение среднего количества товаров в заказе (2.7 → 1.9)
- ✅ **Позитив:** Улучшение логистики: заказов "в ожидании" стало в 2 раза меньше
- 🎯 **Главная рекомендация:** Проверить функционал корзины для 3+ товаров и внедрить систему рекомендаций

## Используемые метрики (14 KPI)
Выручка, AOV, ARPU, маржинальность, конверсия, медианный чек, среднее кол-во товаров в заказе и др.
